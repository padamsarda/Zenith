# Zenith

Zenith is a long-term software project providing a runtime platform for building and
operating an assistant called **Zeni**. Zeni runs on top of the Zenith runtime.

This repository is under incremental, long-term development. The current state is a
minimal project scaffold — no assistant behavior, plugins, or integrations have been
implemented yet.

## Project layout

- `main.py` — runtime entry point.
- `runtime/` — core runtime code.
- `plugins/` — runtime plugins.
- `configs/` — configuration files.
- `architecture/` — architecture documentation and design records.
- `docs/` — general documentation.
- `tests/` — automated tests.

## Requirements

- Python 3.12+

## Running

```bash
python main.py
```
